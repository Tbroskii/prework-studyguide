HTML

CSS

GIT

  • Git is used to help make websites through an iterative process
  • All commands used for GIT start with git and then the command follows
  • GIT allows you to create branches so that you don't have to work off of main

Javascript

  • Javascript is used to make your websites interactive
  • Javascript is a soft typed language meaning there are no types like int or double
  • To create a call a function you type the function name followed by parentheses
